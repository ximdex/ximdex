Wait a sec Speedy...

You should run the Ximdex installer before you can use it.

Follow these easy steps and enjoy Ximdex in a few minutes (you can find the whole install process on the INSTALLATION file).

  1. Open a terminal/console and type the path to the Ximdex folder. For example: 
    $ cd /var/www/ximdex_home
  2. Run the Ximdex installer. 
    $ ./XIMDEX_INSTALL.sh -i
  3. The installer ask you about Ximdex's instance name and the user name and group name for the Ximdex folder. 
    The name for your Ximdex instance will be 'ximdex_home'
			
    - Do you want to modify it (y/N)?
    Select a target directory for Ximdex:
    1. /var/www
    2. /var/www/html
    3. None of these. I'll write the target path directly.
    I've got that your Web Server User is: www-data
			
    - Do you want to modify it (y/N)? y
			
    I've got that your Web Server Group is: www-data
			
    - Do you want to modify it (y/N)? y
  4. Now, the script will create all the required Ximdex infraestructure, adding the default information into the new database. You will be asked for the following information: The words between braquets show the value by default 
    $ Database server [localhost]
    $ Database port [3306]
    $ Admin database user [root]
    $ Admin database password (the current database root user password)
    $ Database name [<XIMDEX_DB_NAME>] (the new database name)
    $ Database user [<XIMDEX_DB_USER>] (the new database user)
    $ Database user password (new password for your new database user)
    $ Ximdex host [http://localhost/<XIMDEX_DB_NAME>] (your server URL for Ximdex)
    $ Ximdex path [/var/www/<XIMDEX_HOME>]
    $ Ximdex admin user [ximdex] (the new Ximdex user)
    $ Ximdex admin password (password for Ximdex user)
    $ Select your Ximdex default language choosing betweeen: 
    1. German
    2. English
    3. Spanish
    4. Portuguese
    Would you like to help us sending anonymous information about usage (Y/n)?
  5. After that, you can decide to run the script 1-MoveXimdexToDocRoot.sh as a priviledged user. This script will set owners for ximdex_home folder
  6. Then, you can reload Ximdex's index page, to finish the installation and Ta Daaa. Enjoy Ximdex!
  7. Once the installation has finished, you can also install more Ximdex modules avaliable on '<XIMDEX_HOME>/modules/' typing the following sentence: 
    $ ./install/module.sh install <MODULE_NAME>
